Web Development

Web development is the process of designing and building websites. It may include the creation of web pages, images, and other web content. I use a content management system (CMS) to build a website or make changes to an existing one and help to develop the web pages in the same way.

Responsive Design

Web developers often design their sites with a specific size in mind; mobile, tablet or desktop view. A responsive design means that the site is able to fit any of these sizes. To create this type of site...web developer need to have coding knowledge and have some understanding of basic web development skills.
